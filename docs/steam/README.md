# Steam グラフィックアセット

ref: https://partner.steamgames.com/doc/store/assets/standard

## 共通ルール

- 画像端まで隙間なくアートワークで埋める。黒枠・余白は不可
- ゲームタイトル以外のテキストを含めない（レビュー、受賞歴、セール文言など不可）
- ロゴは背景に対して読みやすくする
- PG-13相当のアートワークにする
- 違反すると Steam のフィーチャーやセールイベントへの掲載対象外になる

## ストアアセット

### Header Capsule — 920x430

- 用途: ストアページ上部、おすすめ欄、Big Picture、Daily Deal
- デザイン: キーアートとロゴ。ゲームの本質を伝える構図にする
- 補足: 基本的に変更しない。期間限定で Artwork Override を使える

### Small Capsule — 462x174

- 用途: 検索結果、トップセラー、新作リスト
- デザイン: ロゴがカプセルの大部分を占めるようにする。120x45 まで縮小されても読めること
- 補足: 他のカプセルと異なるアートワークでもよい。自動で 120x45, 184x69 が生成される

### Main Capsule — 1232x706

- 用途: Steam ストアトップページのカルーセル
- デザイン: キーアートとロゴ。ゲームプレイの雰囲気が伝わるもの

### Vertical Capsule — 748x896

- 用途: シーズンセール、プロモーションページ
- デザイン: 縦長構図。グラフィック重視で、ロゴは読みやすく配置する

### Page Background — 1438x810（任意）

- 用途: ストアページの背景画像
- デザイン: ページコンテンツと競合しないよう控えめにする
- 補足: 未設定の場合、最後のスクリーンショットから自動生成される

## ライブラリアセット

### Library Capsule — 600x900

- 用途: ライブラリの一覧・コレクション表示
- デザイン: キーアートとロゴ。自動で 300x450 が生成される

### Library Header — 920x430

- 用途: ライブラリの「最近のゲーム」セクション
- デザイン: Header Capsule や Library Capsule と統一感を持たせる
- 補足: 未設定の場合、Header Capsule が使われる

### Library Hero — 3840x1240

- 用途: ライブラリ詳細ページの上部背景
- デザイン: テキストを一切含めない。アートワークのみ。Library Logo が上に重ねて表示される
- 補足: 中央 860x380 がセーフエリア。端はクロップされる可能性がある。自動で 1920x620 が生成される

### Library Logo — 1280x720

- 用途: Library Hero の上に重ねて表示されるロゴ
- デザイン: 透過PNGでゲームタイトルのロゴのみ。他のテキスト不可
- 補足: 配置位置を選択できる（左下、中央上、中央、中央下）
