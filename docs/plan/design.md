このドキュメントは、RUINSのゲームシステム、メカニクス、技術仕様を記録するものです。
世界観・ストーリー設定については `lore.md` を参照してください。

## ゲーム概要

### 基本コンセプト

人間と悪魔の融合体を操り、厳しい自然環境と悪魔の敵を相手に生存・脱出を目指すサバイバルローグライク。
ターン制での戦略的な探索、リソース管理、モード切り替えによる判断が特徴です。

### ジャンル・特徴
- **ジャンル**: ローグライク・サバイバルRPG（ターンベース）
- **視点**: 1人称レイキャスティング（3D風2D）
- **探索**: グリッドベース・ターン制移動
- **戦闘**: 同一マップ上でのターンベース統合戦闘
- **成長**: ラン内での使用ベーススキル成長
- **言語**: 完全日本語対応

### 参考作品

#### Cataclysm: Dark Days Ahead (CDDA)
**参考にする要素:**
- **Action Points (AP) システム**: 敏捷性ベースのAP獲得、債務システム
- **地形コスト**: 移動コストの地形別差別化
- **装備重量影響**: 重装備による移動・攻撃コスト増加
- **サバイバル要素**: 食料、体温管理

#### The Long Dark
**参考にする要素:**
- **体温管理**: 寒冷環境での生存
- **リソース管理**: 有限のリソースでの判断
- **環境の脅威**: 自然そのものが敵

## レベル構造

### 大域移動: ハブ型

```
           [廃坑 10F]
         /
[ハブ] ──┼─ [研究所 15F]
         \
           [深淵 20F]
          ...
```

- ハブから解禁済みのダンジョンに出発する
- ダンジョンの数と深さはダンジョンごとに異なる
- ダンジョンをクリアすると次のダンジョンが解禁される
- ハブで通貨を使って買い物し、準備してから出発する

### ダンジョン設計原則

- ダンジョンごとに環境、敵種、有効な戦術が異なる
- ダンジョン数と深さはデータ追加で拡張できる
- 最深部にボスがいる

### ダンジョン内: 階層選択

```
[1F]
  ↓ 選択肢提示
 ┌┴┐
[2A][2B]  ← どちらかを選ぶ
  ↓
 ...
  ↓
[最深部 ボス]
```

- 各階層クリア後、次の階層の選択肢が提示される
- 選んだ方に進む（戻れない）
- 選ばなかった方は行けない
- 最深部でボス戦

### 小域探索

各階層内での自由探索。

- グリッドベースのマップ
- ターン制移動
- リソース収集、敵との戦闘

## ランサイクル

「ラン」はハブを出発してからダンジョンを終了するまでの1回の探索を指す。
スキル・装備・アイテムは毎ラン完全リセットされる。永続するのは通貨とダンジョン解禁実績のみ。

### ランの流れ

```
1. ハブにスポーン（職業の初期装備を所持した状態）
      ↓
2. 店で追加の装備・アイテム・本を購入、合成でカスタム
      ↓
3. ダンジョンを選択
      ↓
4. ダンジョンに入る（マップがランダム生成される）
      ↓
5. 階層を探索（敵と戦闘、アイテム収集、スキル成長）
      ↓
6. 次の階層を選択
      ↓
7. 繰り返し...
      ↓
8. 最深部ボス撃破 → 所持アイテム自動売却、通貨獲得、次のダンジョン解禁
   or 撤退 → 所持アイテム自動売却、通貨の一部獲得
   or 死亡 → 所持アイテム消滅、通貨獲得なし
      ↓
9. ハブに戻る（初期装備に戻った状態で再スポーン）
```

### マップ生成

- ダンジョンに入るたびにマップが新規にランダム生成される
- 一度探索した階層に戻ることはできない
- クリア済みのダンジョンにも再挑戦できる

### 通貨システム

- ダンジョンクリア・撤退時に所持アイテムが自動売却され、通貨に変換される
- クリア時はボーナス通貨も獲得する
- 撤退時は到達階層に応じた通貨を獲得する
- 死亡時は所持アイテムが消滅し、通貨を獲得できない
- ハブの店で通貨を使って追加の装備・アイテム・本を購入してからランに挑む
- 何を買って持ち込むかがランの準備フェーズの判断になる

### 永続解禁（将来実装）

通貨を使って段階的に解禁できる永続要素。プレイを続ける動機になる。

- **店売り商品の追加** — 購入可能な装備・アイテム・本の品揃えが増える
- **職業の追加** — 新しい職業が選択可能になる
- **特質の追加** — 選択可能な特質が増える
- **レシピ解禁** — 合成で作れるものが増える
- **ハブ施設の拡張** — 鍛冶場、図書館などの施設を建設し、ラン準備の選択肢が広がる
- **初期装備のアップグレード** — 職業の初期装備をより良いものに置き換える
- **ダンジョン情報の開示** — 未踏ダンジョンの環境・敵種のヒントを事前に見られる
- **初期スキルボーナス枠** — ラン開始時のスキル値に+1できるスロットを購入

### 永続する要素と一時的な要素

| 要素 | 永続性 | 説明 |
|------|--------|------|
| 通貨 | 永続 | ダンジョンクリア・撤退で獲得する |
| ダンジョン解禁実績 | 永続 | クリアしたダンジョンの記録 |
| 職業・特質 | 永続 | ハブで通貨を支払って変更できる |
| スキル値 | ラン内 | 毎ラン0から成長する |
| 装備・アイテム | ラン内 | 帰還時に自動売却され初期装備に戻る |
| ダンジョンのマップ | ラン内 | 入るたびに新規生成 |
| ダンジョン内の進行 | ラン内 | 撤退・死亡で1Fからやり直し |

## モードシステム

### 人間モード

通常状態。

| 項目 | 値 |
|-----|-----|
| 戦闘力 | 通常 |
| 食料消費 | 1倍 |
| 体温変化 | 環境に依存 |
| 特殊能力 | なし |

### 悪魔モード

戦闘強化状態。

| 項目 | 値 |
|-----|-----|
| 戦闘力 | 強化（2倍程度） |
| 食料消費 | 2倍 |
| 体温変化 | 上昇（+0.5℃/ターン） |
| 特殊能力 | 悪魔スキル使用可能 |

### 起動条件

- 体温35℃以上で起動可能
- 低体温状態では起動できない

### 終了条件

- 任意で解除
- 体温40℃到達で強制解除（過熱リスク）
- 食料切れで強制解除

## サバイバルシステム

### 体温

```
体温ゲージ: 25℃ - 42℃

 25℃: 凍死
 30℃: 重度低体温（行動不能）
 33℃: 低体温（行動ペナルティ）
 35℃: 悪魔モード起動可能ライン
 36℃: 平熱
 38℃: 悪魔モード中の通常体温
 40℃: 過熱警告
 42℃: 過熱死
```

### 体温変化要因

| 要因 | 変化 |
|-----|------|
| 寒冷環境 | -0.5〜-2℃/ターン |
| 焚き火 | +1℃/ターン |
| 悪魔モード | +0.5℃/ターン |
| 移動 | +0.2℃/ターン |
| 防寒装備 | 低下を軽減 |

### 食料管理

```
食料ゲージ: 飢餓 ~ 満腹

人間モード: -1%/ターン
悪魔モード: -2%/ターン

0%: 餓死開始（体温も低下）
```

## スキルシステム

### 使用ベースの成長

RPG的な経験値・キャラクターレベルは存在しない。
スキルを使えばそのスキルが伸びる方式で、毎ランゼロから成長する。
全36スキルが7カテゴリに分類される。詳細は `docs/design/20260319_13.md` を参照。

### 本による成長

ダンジョン内で本を拾うか、ハブの店で本を購入して持ち込める。
本を読むとスキル経験値を獲得できる。実戦では伸ばしにくい知識系スキルは本が主要な成長手段になる。
読書にはAPを消費し、読了した本は消滅する。

### 職業

ゲーム開始時に職業を選択する。
職業はラン開始時のスキル初期値と初期装備を決定する。
ラン中に別方向にスキルを育てることも可能。

## ターンベース探索システム

### 基本仕様
- **グリッド移動**: 1マス単位での移動
- **アクションコスト**: 全行動に時間コストを設定
- **ターン制**: プレイヤー行動→敵行動→環境変化
- **統合戦闘**: 探索マップ上で直接戦闘実行

### アクションコストシステム

**基本メカニズム:**
- **APの獲得**: 毎ターン開始時に「敏捷性」値分のAPを獲得
- **基本移動コスト**: 通常地形での1マス移動 = 100AP
- **ターン終了条件**: APが0以下になるとターン終了

**行動コスト例:**
```
基本行動:
- 移動（平地）: 100AP
- 移動（対角）: 142AP（√2倍）
- 通常攻撃: 100AP
- 悪魔スキル: 150-300AP
- アイテム使用: 80-120AP
- モード切り替え: 50AP
```

## 難易度設計

### ダンジョン別難易度

各ダンジョンは独自の環境と難易度を持つ。
階層が深くなるほどダンジョン内でも難易度が上昇する。

ダンジョンの具体的な構成（名称、階層数、環境、敵種）は別途定義する。

### 難易度の軸

| 軸 | 浅いダンジョン | 深いダンジョン |
|----|--------------|--------------|
| 環境温度 | 穏やか | 極端（極寒/灼熱） |
| 敵の強さ | 弱い | 強い |
| 必要装備 | 基本装備で可 | 高性能装備が必須 |
| 有効な戦術 | 汎用的 | 特化が求められる |

### バランス方針

- 悪魔モードなしでもギリギリ生存可能
- 悪魔モードで戦闘が有利
- 高難度ダンジョンは悪魔モード or 高装備が必須
- 通貨を貯めて準備を整え、難しいダンジョンに挑むプレイが成立する
- 死亡すると通貨を獲得できないため、無謀な挑戦にはリスクが伴う
