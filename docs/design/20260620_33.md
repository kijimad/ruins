# Stable Diffusion による Steam グラフィック生成手順

## 背景

Steam ストアグラフィックの背景画像を、写真素材から Stable Diffusion (img2img) でピクセルアート風に変換して生成する。洞窟写真をベースに、凍晶の遺跡の雰囲気を持つゲームアート風画像を作る。

## 設計

### 環境構築

RTX 3050 6GB 環境を前提とする。SD Turbo (stabilityai/sd-turbo) は VRAM 6GB で動作する軽量モデル。

```bash
# nix の python で venv を作成
nix shell nixpkgs#python312 --command python3 -m venv /tmp/sd_venv
source /tmp/sd_venv/bin/activate

# PyTorch (CUDA 12.4) と diffusers をインストール
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
pip install diffusers transformers accelerate pillow
```

### 実行時の環境変数

nix 環境では libstdc++ と NVIDIA ドライバーライブラリのパスを明示する必要がある。

```bash
# libstdc++ を nix store から取得
export LD_LIBRARY_PATH="$(find /nix/store -maxdepth 3 -name 'libstdc++.so.6' -path '*gcc*lib*' 2>/dev/null | head -1 | xargs dirname)"

# NVIDIA ドライバーライブラリのシンボリックリンクを作成
mkdir -p /tmp/nvidia_libs
ln -sf /usr/lib/x86_64-linux-gnu/libcuda.so.1 /tmp/nvidia_libs/
ln -sf /usr/lib/x86_64-linux-gnu/libcuda.so /tmp/nvidia_libs/
find /usr/lib/x86_64-linux-gnu -name "libnvidia*.so*" -exec ln -sf {} /tmp/nvidia_libs/ \; 2>/dev/null

export LD_LIBRARY_PATH="$LD_LIBRARY_PATH:/tmp/nvidia_libs"
```

### 生成スクリプト

```python
import torch
from diffusers import AutoPipelineForImage2Image
from PIL import Image

pipe = AutoPipelineForImage2Image.from_pretrained(
    "stabilityai/sd-turbo",
    torch_dtype=torch.float16,
    variant="fp16",
)
pipe = pipe.to("cuda")
pipe.enable_attention_slicing()

def pixelate(image, pixel_size):
    """画像をピクセルアート化する。縮小してからニアレストネイバーで拡大する"""
    w, h = image.size
    small = image.resize((w // pixel_size, h // pixel_size), Image.NEAREST)
    return small.resize((w, h), Image.NEAREST)

img = Image.open("docs/steam/dungeon.jpg").convert("RGB")

# クロップしてアセットサイズにリサイズ
cropped = img.crop((0, 600, 5767, 3300)).resize((920, 432), Image.LANCZOS)

# 入力をピクセルアート化してからimg2imgに渡すとピクセル感が強まる
pixelated = pixelate(cropped, 4)

result = pipe(
    prompt="16-bit pixel art, retro game screenshot, dark frozen ancient stone dungeon corridor, bright blue glowing ice crystals on walls, warm orange torchlight, fantasy RPG dungeon, limited color palette, chunky pixels, SNES style",
    negative_prompt="realistic, photo, 3d render, smooth, high resolution, modern, blurry, text, watermark, person",
    image=pixelated,
    strength=0.75,
    guidance_scale=2.0,
    num_inference_steps=4,
).images[0]

# 出力もピクセルアート化で仕上げる
result_pixel = pixelate(result, 3)
result_pixel.save("output.png")
```

### パラメータの調整指針

| パラメータ | 効果 |
|-----------|------|
| `strength` | 0.5: 元写真に近い。0.8: 大きく変わる。0.65-0.75 が実用的 |
| `pixelate` の入力 pixel_size | 大きいほど粗いピクセルアートになる。4 で良好 |
| `pixelate` の出力 pixel_size | 3 で仕上げのピクセル感を付与 |
| `num_inference_steps` | SD Turbo は 1-4 ステップで動作する。4 で十分 |
| `guidance_scale` | SD Turbo は 1.0-2.0 を推奨。大きくするとプロンプトに忠実 |

### 採用プロンプト (v4)

v4 はリアル寄りの生成 + scale 2 の決定論的ピクセルアート化で、Steam アセットに適した高精細ピクセルアートを得る。v3 の scale 3 は粗すぎ、ai_r シリーズの scale 4 は変化が小さすぎた。

共通設定:
- model: `stabilityai/sd-turbo` (fp16)
- `strength`: 0.75
- `guidance_scale`: 2.0
- `num_inference_steps`: 4
- 後処理: `pixelate_lanczos(result, 2)` → `darken(result, 0.6)`
- 入力のピクセルアート化は行わない

```
prompt:
dark ancient stone dungeon corridor, blue glowing ice crystals growing on the walls only, crystal clusters attached to stone walls, no crystals on the floor, warm torch light hanging from ceiling above the archway, clean stone floor, cold dark atmosphere, fantasy dungeon, dramatic lighting, detailed, high quality

negative_prompt:
blurry, text, watermark, person, character, bright, overexposed, crystals on floor, ice on ground
```

ピクセルアート化関数（v4 で使用）:
```python
def pixelate(image, scale):
    """LANCZOS shrink → NEAREST expand。scale が小さいほど高精細"""
    w, h = image.size
    small = image.resize((w // scale, h // scale), Image.LANCZOS)
    return small.resize((w, h), Image.NEAREST)
```

v3 との違い:
| | v3 | v4 |
|--|----|----|
| プロンプト | ピクセルアートを指示 | リアル描写を指示 |
| strength | 0.65 | 0.75 |
| ピクセルアート化 | `pixelate(result, 3)` NEAREST→NEAREST | `pixelate(result, 2)` LANCZOS→NEAREST |
| 解像度感 | 粗い（1/3 縮小） | 高精細（1/2 縮小） |

## コメント

- SD Turbo は軽量で高速だが、品質は SDXL に劣る。VRAM に余裕があれば SDXL Turbo や FLUX も検討する
- seed を固定すると再現可能。`generator=torch.Generator("cuda").manual_seed(42)` を pipe に渡す
- 生成した背景は Photopea でロゴと組み合わせて最終アセットに仕上げる
