# アイテムに重さを追加する

## 背景

スカベンジングをこのゲームの醍醐味とするが、持てる重量を制約することで「どれを持ち帰るか」あるいは「何を持ち込むか」の選択をさせたい。また、パワーアップのモチベーションとしたい。

## 設計

- すべての所持アイテムは重量(kg)を持つ
- 所持アイテムあるいはステータスに変化があったときに重量の再計算を行わせる
- 重さを超えて持てるが、悪いペナルティがつく。ペナルティは空腹度のように段階的に変化する
- 所持可能重量は、「筋力」ステータスに依存する

## ステップ

1. アイテムに重さコンポーネントを付与する
2. キャラクタに所持可能重量をつけ、ステータスに変化があったときに更新させる
3. 重さによるペナルティをつける

## 例

- バット(1kg) 武器
- 純金の時計(0.5kg) お宝
- 水晶玉(2kg) お宝
- 新聞紙(0.1kg) 燃料
- 回復薬(0.01kg) 使用

## 進捗

- [x] すべての所持アイテムが重量を持つ。`internal/components/components.go` の `Weight` コンポーネント、`assets/metadata/entities/raw/raw.toml` で全アイテムに `weight` を指定、`internal/raw/raw.go` の `ParseWeight` で解釈する
- [x] アイテムに重さコンポーネントを付与する。`gc.Weight` を `EntitySpec` 経由で付与、`internal/components/components_gen.go` に登録済み
- [x] キャラクタに所持可能重量をつける。`internal/components/components.go` の `WeightCapacity`、`internal/world/query/weight.go` の `UpdateWeightCapacity`
- [x] 所持可能重量は筋力ステータスに依存する。`internal/world/query/weight.go` の `calculateMaxCarryingWeight` が `Abilities.Strength` から算出する
- [x] 所持アイテムあるいはステータスに変化があったときに重量を再計算する。`internal/components/components.go` の `WeightDirty` フラグ、`internal/systems/weight_dirty.go` の `WeightDirtySystem` が再計算、`internal/systems/stats_changed.go` がステータス変化時に、`internal/world/lifecycle/location.go` がアイテム移動時にフラグを付与する
- [x] 重さを超えて持てるが段階的なペナルティがつく。`internal/world/query/turn.go` の `calculateOverweightPenalty` が超過量に応じた Speed ペナルティを付与し、`internal/activity/move.go` は最大の1.5倍超で移動不能とする
