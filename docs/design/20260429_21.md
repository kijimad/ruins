# オープニングナレーション案

## 方針

- 手塚治虫風の俯瞰的・突き放したナレーション
- 語り手に人格を持たせる。人間を観察して呆れている視点
- 短い文で畳みかける
- ナレーション→生の会話の切り替えで締める

## テキスト

### page1a（黒背景）

```
見渡すかぎりの荒野に、大穴がひとつ、口を開けている。
```

### page1b（街画像）

```
穴の底には古代文明の遺跡がある。
宝が出る。怪物も出る。潜った者の半分は帰ってこない。
穴のまわりには潜る者、売る者、買う者で街ができた。
```

### page2（酒場背景）

```
「最下層を目指したやつ、また死んだらしいな。」

「たまにいるんだよな、そういう命知らずが。」
「その前のやつも、その前のやつも死んだ。」

「でさ、また来たらしいぜ。新しい[拾い屋]が。」
```

## 演出ノート

- page1a: 一文だけ。映像に任せる
- page1b: 「帰ってきたやつがいないからだ」で伝説にアイロニーを入れ、「人間というやつは」で手塚的な突き放した語りにする
- page2: ナレーションから一転、生の会話。プレイヤーの存在を第三者の噂として導入する
- [拾い屋] はキーワードハイライト表示
